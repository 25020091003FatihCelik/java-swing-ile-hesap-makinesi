##Java Swing ile Hesap Makinesi

Bu proje, Java Swing kullanılarak geliştirilmiş basit bir masaüstü hesap makinesi uygulamasıdır.
Proje, temel aritmetik işlemleri gerçekleştirebilen kullanıcı dostu bir arayüz sunmaktadır.

##🚀 Proje Hakkında

Uygulama aşağıdaki temel matematiksel işlemleri gerçekleştirebilmektedir:

➕ Toplama

➖ Çıkarma

✖️ Çarpma

➗ Bölme

Kullanıcı, arayüz üzerinden sayıları girerek işlem seçebilir ve sonucu anında görüntüleyebilir.

##🛠️ Kullanılan Teknolojiler

Java

Java Swing (GUI)

NetBeans / IntelliJ (Projeye göre düzenleyebilirsin)

##📂 Proje Yapısı

src/ → Kaynak kod dosyaları

build.xml → Proje yapılandırma dosyası

manifest.mf → Uygulama manifest dosyası

##▶️ Uygulamayı Çalıştırma

Projeyi bilgisayarınıza klonlayın:

git clone https://github.com/kullaniciadi/java-swing-ile-hesap-makinesi.git

IDE (NetBeans / IntelliJ) ile projeyi açın.

Ana sınıfı çalıştırarak uygulamayı başlatın.

##🎯 Projenin Amacı

Bu proje ile:

Java Swing arayüz geliştirme mantığını öğrenmek

Olay tabanlı programlama (Event Handling) yapısını kavramak

Temel Java bilgilerini pekiştirmek

amaçlanmıştır.
